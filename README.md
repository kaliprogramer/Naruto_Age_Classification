# 🍥 Naruto Age Classifier

A Streamlit web app that predicts the age group of Naruto characters from uploaded images using a Convolutional Neural Network (CNN). This project demonstrates computer vision with PyTorch in a fun and interactive way for anime fans.  

**Demo:** [https://narutoai.streamlit.app/](https://narutoai.streamlit.app/)  

---

## 🔍 Features

- Upload your own character image or use a sample.
- Predicts the age group: `Child`, `Teen`, or `Adult`.
- Displays prediction confidence and probability bars.
- Simple, visually appealing UI with animated cards.
- Compatible with CPU and GPU for fast inference.

---

## 🛠️ Technologies Used

- **Python 3.11**
- **Streamlit** – for web interface
- **PyTorch** – for CNN model
- **Torchvision** – for image preprocessing
- **PIL / Pillow** – for image handling
- **NumPy** – for probability calculations
- **Requests** – for fetching sample images
